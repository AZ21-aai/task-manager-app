# 🚀 Task Manager App (Fullstack)

Eine moderne Fullstack Task-Manager Anwendung zur Verwaltung von Aufgaben.

## 🧩 Tech Stack

### Backend
- Java 17
- Spring Boot
- REST API
- PostgreSQL
- JPA / Hibernate

### Frontend
- React (Vite)
- JavaScript (ES6)
- CSS (modern UI)

---

## ⚙️ Features

- ✅ Aufgaben erstellen
- ✅ Aufgaben als erledigt markieren
- ✅ Aufgaben löschen
- ✅ Status wird gespeichert (Backend + DB)
- ✅ Moderne UI mit Hover-Effekten
- ✅ REST API Integration

---

## 🔗 API Endpoints

| Methode | Endpoint            | Beschreibung          |
|--------|--------------------|----------------------|
| GET    | /api/tasks         | Alle Tasks laden     |
| POST   | /api/tasks         | Task erstellen       |
| PUT    | /api/tasks/{id}    | Task aktualisieren   |
| DELETE | /api/tasks/{id}    | Task löschen         |
